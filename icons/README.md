# Workly Extension Icons

This folder contains the required icons for the Chrome extension.

## Required Icons:
- icon16.png (16x16) - Toolbar icon
- icon32.png (32x32) - Windows taskbar
- icon48.png (48x48) - Extension management page
- icon128.png (128x128) - Chrome Web Store

## Icon Design:
The icons should feature a clock or time-related symbol to represent the concept of work hours calculation. Use a clean, modern design with good contrast.

For now, you can use placeholder icons or create simple ones using any image editor. The extension will work without icons, but they improve the user experience.

## Colors:
- Primary: #667eea (Blue gradient start)
- Secondary: #764ba2 (Purple gradient end)
- Background: White or transparent
